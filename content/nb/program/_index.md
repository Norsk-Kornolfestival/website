## Program 2026

Festivalen går over fire dager, fra torsdag 1. oktober til søndag 4. oktober. Hovedfestivalen er fredag og lørdag.

### Torsdag 1. oktober

| Tid | Hva |
|-----|-----|
| ca. 16:50 | Buss fra Vigra (Ålesund) flyplass til Hornindal |
| 19:30–22:00 | **Oppskåke** — åpningsfeiring i historisk gårdsbygning (egen billett). Oppmøte på Raftevold Hotel. |

### Fredag 2. oktober

| Tid | Hva |
|-----|-----|
| 10:00–16:00 | **Tradisjonell gårdsbrygging** i eldhus fra 1600-tallet. Ledet av Stig Seljeset (kveik #22 Stålljen). Einer-infusjon, mesking, separering, kjøling, kveikpitching med tradisjonelt skriking. Ingen koking (rå-øl). Inkluderer spekemat, flatbrød-lunsj og kornøl. *(Egen billett: Bryggepass)* |
| 11:00–15:00 | **Bryggeøkt kun for kvinner** *(egen billett)* |
| 17:00 | Dørene åpner |
| 18:00 | Offisiell festivalåpning |
| 20:00 | Hjemmebryggerene pakker sammen; kommersielle bryggerier begynner salg |
| 00:30 | Festivalen stenger |

### Lørdag 3. oktober

| Tid | Hva |
|-----|-----|
| 11:30 | Valgfri guidet tur til Nakkebruna-hytta (190 m stigning, ca. 30 min). Valgfri utvidelse til steinbrudd fra neolittisk/bronsealder. |
| 12:00 | Festivalen åpner |
| 12:00 | **Øystein Menes** demonstrerer ølbrygging (Sogn-tradisjon, 40 års erfaring) |
| 12:00 | **Rolf Taraldset** (trekunstner) demonstrerer tradisjonell ølbolleskjæring |
| 14:30 | **Geir Grønnesby**: «Hva er sammenhengen mellom gårdsetableringa og ølbrygginga?» — arkeologiske bryggstein, jernalder |
| 17:00 | **Xavi Tortosa**: «Tradition Reimagined: Applying Brewing Heritage in Modern Craft» |
| 18:00 | Hjemmebryggerene pakker sammen; kommersielle bryggerier fortsetter |
| 18:45 | **Festivalmiddag** på Raftevold Hotel — saudehaud, kokte poteter, kålrotstappe, Nordfjord-pannekaker. Tre stiler tradisjonelt øl: Hornindal konnjøl, Voss heimabrygg, Stjørdalsøl. *(Egen billett)* |
| 19:00 | **Christina Wade**: «Drinking with the Dead: Women, Ale, and Ritual in Viking-Age Ireland» |
| 21:00 | Konkurransevinnere annonseres |
| 01:00 | Festivalen stenger |

### Tidspunkt ikke fastsatt

- **Jan Steensels** (VIB-KU Leuven): «Analyse av 1760 stammer av gårdsbryggingsgjær» — upublisert stor genetisk studie
- **Lars Marius Garshol**: «Middelalderens vannmyte» — drakk folk øl fordi vannet var utrygt?

### Søndag 4. oktober

| Tid | Hva |
|-----|-----|
| 09:30 | Buss til Ålesund flyplass (kobler til 13:20-flyet) |
